# 💰 Expense Tracker

A simple command-line Expense Tracker built using Python. This application helps users manage daily expenses, search records, calculate total spending, and store data permanently using file handling.

---

## ✨ Features

- ➕ Add Expense
- 📋 View Expenses
- 🔍 Search Expense
- ❌ Delete Expense
- 💵 Calculate Total Spending
- 💾 Save Expenses to File
- 📂 Load Saved Expenses Automatically

---

## 🛠 Technologies Used

- Python 3
- File Handling
- Lists
- Dictionaries

## ▶️ How to Run

1. Install Python 3.
2. Open the project folder.
3. Run:

```bash
python main.py
```
